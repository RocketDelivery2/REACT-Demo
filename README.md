## Tech Stack

- **Frontend:** React (Create React App)
- **Tooling:** CRA scripts (Webpack/Babel under the hood)

---

## Prerequisites

- **Node.js** (LTS recommended)
- **npm**

---

## Install

```bash
npm install
```

---

## Run Locally (Development)

```bash
npm start
```

Then open: http://localhost:3000

---

## Test

```bash
npm test
```

---

## Build

```bash
npm run build
```

Build output is generated in the `build/` folder.

---

## Eject (advanced)

```bash
npm run eject
```

> This is a one-way operation. Only use it if you need full control over the underlying Webpack/Babel configuration.
